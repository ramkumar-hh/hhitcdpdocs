## 1. Executive Summary
Vesper Analytics is a scalable, multi-tenant Software-as-a-Service (SaaS) platform designed to deliver secure, branded analytics portals. By leveraging embedded Microsoft Power BI, the platform allows organizations to provide tailored data insights to their internal teams and external clients without exposing the underlying BI infrastructure.

## 2. Platform Capabilities
### 2.1 Multi-Tenant Architecture
Each client organization (tenant) operates in a logically isolated environment. Tenants have custom branding, distinct user bases, and dedicated navigation hierarchies. Data isolation is strictly enforced to prevent cross-tenant exposure.

### 2.2 Role-Based Access Control (RBAC)
Vesper implements a granular access control system based on Groups, Roles, and Permissions:
- **Groups:** Logical collections of users (e.g., departments, clients).
- **Roles:** Defined sets of permissions (view, edit, delete capabilities).
- **Permissions:** Actions mapped to specific platform resources.
This ensures users only see the reports and data segments they are strictly authorized to view.

### 2.3 Identity and Authentication
The platform supports multiple authentication strategies:
- **Local Authentication:** Standard email and password login with mandatory strong hashing and optional Multi-Factor Authentication (MFA).
- **Enterprise SSO:** Native integration with Microsoft Entra ID for seamless enterprise access and automated group mapping.

### 2.4 Embedded Power BI
At its core, Vesper securely embeds Power BI reports directly into the portal. The platform handles all authentication against Microsoft's services natively, meaning end-users do not require individual Power BI licenses or direct access to the Power BI workspace.

### 2.5 File Uploads and Storage Integration
Authorized users can upload data files directly through the Vesper portal into their organization's Microsoft SharePoint libraries or OneLake storage, streamlining the analytics ingestion workflow.

## 3. Subscription Tiers
Vesper Analytics is adaptable to varying organizational needs:
- **Standard:** Core portal features, RBAC, and Power BI embedding.
- **Professional:** Adds Enterprise SSO, custom integrations, and advanced configurations.
- **Enterprise:** Full platform capabilities with dedicated support and custom data isolation strategies.

## 4. Security Highlights
Security is foundational to the Vesper platform:
- All sensitive operations are heavily audited.
- Sessions are secured via short-lived tokens.
- Cross-tenant data isolation is guaranteed at the database level.
